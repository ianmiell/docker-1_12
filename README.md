Docker 1.12 Demo
================

An automated, annotated, interactive demo of Docker 1.12's new features using [ShutIt](https://github.com/ianmiell/shutit/blob/master/README.md).

```
pip install shutit
git clone --recursive https://github.com/ianmiell/docker-1_12
cd docker-1_12
./run.sh
```
